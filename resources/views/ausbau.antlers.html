{{ partial:layout/section class="bg-cloud" }} {{ partial:layout/inner }} {{ partial:components/headings/h1 title="Ausbau" }} {{ partial:layout/grid }} {{ partial:layout/span }} {{ partial:components/headings/h3 title="Bodenbeläge" }}

Boden in Wohnung: Klötzliparkett (Monopark) Eiche, matt versiegelt
Nasszellen (Küchen und Waschküche UG): Feinsteinzeugplatten, unglasiert, SABAG Modico, anthrazit

{{ partial:components/headings/h3 title="Wandbeläge und Decken" }}

Wände: Raufasertapete weiss gestrichen, in den Nasszellen wo keine Platten vorhanden sind, Weissputz, weiss gestrichen
Decken: Weissputzdecken, weiss gestrichen

{{ partial:components/headings/h3 title="Küche" }}

Bodenbeläge: Feinsteigzeugplatten, SABAG Modico, anthrazit
Rückwand: Feinsteigzeugplatten, SABAG Modico, hellgrau
Decke: Weissputz, weiss gestrichen
Arbeitsplatte: Granit poliert, Steel Gray Küchenfronten/Küchenschränke: Kunstharz, Sandbeige seidenmatt
Geräte: Glaskeramik, Dunstabzug (Flachschirmhaube), Geschirrspüler, Kühl- und Gefriergerät, Marke Siemens

{{ partial:components/headings/h3 title="Badezimmer" }}

Bodenbeläge: Feinsteigzeugplatten, SABAG Modico, anthrazit
Wände: Feinsteigzeugplatten, SABAG Modico, beige
Decke: Weissputz, weiss gestrichen
Sanitärapparate: Keramik weiss, Spiegelschrank weiss
Armaturen: Chromeline
Duschwanne mit Vorhangschiene oder Badewanne

{{ partial:components/headings/h3 title="Aussenfläche" }}

Boden Sitzplatz: Zementplatten
Boden Balkon: Betonplatte, Monobeton, sicht
Balkongeländer: Hohlprofile Metall, pulverbeschichtet

{{ partial:components/headings/h3 title="Fenster" }}

Kunststofffenster: dreifach verglast, innen weiss, aussen mit Farbfolie anthrazit grau, Vorhangschiene

{{ partial:components/headings/h3 title="Storen und Sonnenschutz" }}

Fenster: Lamellenstoren, bronze, zum Teil Rollladen, bronze, Handantrieb
Balkone: Gelenkarmmarkisen ohne Volant, grau, Handkurbel

{{ partial:components/headings/h3 title="Elektroanlagen" }}

Sonnerie: Gegensprechanlage
Küchenmöbel: Einbauspots
Glasfaser: in jeder Wohnung, Mieter wählt selbstständig zwischen den Anbietern und schliesst ein Abonnement ab
Mehrere dreifach Steckdosen und Schalter in den Räumen, Multimedia-Verteiler im Eingangsbereich

{{ partial:components/headings/h3 title="Heizungs- und Lüftungsanlagen" }}

Beheizung mittels Heizkörper, Gasheizung
Die Abluft (Küche und Nasszellen) erfolgt über Dach. Frischluftzufuhr über Aussenluftdurchlass über den Fenstern

{{ partial:components/headings/h3 title="Liftanlagen" }}

Pro Haus ein Personenaufzug für maximal sechs Personen

{{ partial:components/headings/h3 title="Keller" }}

Pro Wohnung ein Kellerabteil, mit Steckdose auf Wohnungszähler, Schliessung mit Zylinder auf Wohnungsschliessung

{{ partial:components/headings/h3 title="Waschküche und Trocknungsraum" }}

Jede Wohnung verfügt im Untergeschoss über eine Waschmaschine und einen Trockner, Marke Siemens. Trocknungsraum im UG pro Haus mit Secomat

{{ partial:components/headings/h3 title="Veloabstellplätze" }}

Veloräume mit zahlreichen Abstellplätzen, mögliche Ladestationen für E-Bikes (Steckdosen)

{{ partial:components/headings/h3 title="Entsorgung" }}

Vier Container in der Überbauung, Abstellflächen für Karton- und Zeitungslagerung vorhanden

{{ partial:components/headings/h3 title="Parking" }}

35 Einstellplätze davon 27 Einstellplätze mit Option E-Ladestation (Flachband für E-Mobilität vorbereitet)

{{ partial:components/headings/h3 title="Bastelräume" }}

Vier Bastelräume im UG

{{ partial:components/headings/h3 title="Umgebung" }}

Begrünte Aussenanlage mit Sitzbänken, Spielplatz und Steingrill
